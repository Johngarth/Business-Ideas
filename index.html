One-time setup (board owner only)

This board syncs live through a free Supabase project you control. Five minutes:

  1. Create a free project at supabase.com (any name, any region).
  2. In the project: SQL Editor → run this once: 
    create table boards (
  id text primary key,
  data jsonb not null default '{}',
  updated_at timestamptz default now()
);
alter table boards enable row level security;
create policy "open board" on boards
  for all using (true) with check (true);
alter publication supabase_realtime add table boards;
insert into boards (id, data) values
  ('main', '{"title":"Untitled board","notes":[],"links":[]}');
  3. Project Settings → API: copy the Project URL and the anon public key.
  4. Edit this file, find CONFIG at the top of the script, paste both values.
  5. Host the file (GitHub Pages or Netlify Drop) and share the URL.

Note: anyone with the page URL can read and edit the board — the link is the key.
